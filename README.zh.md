## 这是什么

为独立创业者整理的 toolkit + operator playbook——给那些想在没有团队的情况下并行发布多个 AI 产品的人。

**这不是一个 fork。** 栈中每一个 skill、agent 和 prompt 都标注了原作者。请参阅 [stack/](stack/) 和 [credits/](credits/)。

## 技术栈概述

**核心（必装）:**
- Claude Code（运行时）
- Obsidian（`~/Brain` vault）
- graphify（知识图谱层）
- Frontend-Design（独特 UI 生成）

**可选:**
- Everything Claude Code（skills + agents — 想要广泛目录时添加）
- Toprank（SEO + 广告 — 做 SEO 或付费广告时添加）

```bash
git clone https://github.com/mccarthy606/claude-operator-stack.git
cd claude-operator-stack
./install.sh --dry-run
```
