## これは何か

ソロファウンダー向けのキュレーションされたツールキット + オペレータープレイブック — チームなしで複数の AI プロダクトを並行してリリースしたい人のために。

**これはフォークではありません。** スタック内のすべての skill、agent、prompt はオリジナルの作者をクレジットしています。 [stack/](stack/) と [credits/](credits/) を参照してください。

## スタック概要

**コア（常に入れる）:**
- Claude Code（ランタイム）
- Obsidian（`~/Brain` vault）
- graphify（ナレッジグラフ層）
- Frontend-Design（独特な UI 生成）

**オプション:**
- Everything Claude Code（skills + agents — 広いカタログが必要になったら追加）
- Toprank（SEO + 広告 — SEO や有料広告をやる場合に追加）
