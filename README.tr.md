## Bu nedir

Tek başına çalışan kurucular için curated toolkit + operatör playbook'u — birden fazla AI ürününü ekipsiz, paralel olarak yayınlamak isteyenler için.

**Bir fork değil.** Stack'teki her skill, agent ve prompt orijinal yazarına atıf yapar. [stack/](stack/) ve [credits/](credits/) bölümlerine bakın.

## Stack özeti

**Çekirdek (her zaman kur):**
- Claude Code (runtime)
- Obsidian (`~/Brain` vault)
- graphify (bilgi grafı katmanı)
- Frontend-Design (özgün UI üretimi)

**İsteğe bağlı:**
- Everything Claude Code (skills + agents — geniş katalog için ekle)
- Toprank (SEO + Reklam — SEO veya ücretli reklam yapıyorsanız ekle)

```bash
git clone https://github.com/mccarthy606/claude-operator-stack.git

./install.sh --dry-run

```
